# 交付质量检查清单

## A. 主干匹配
- [ ] 已确认软件类型（平台/分析/工具）
- [ ] 主干结构与类型一致
- [ ] 未确认类型时未提前输出完整正文

## B. 结构完整性
- [ ] 一级章节完整
- [ ] 关键二级章节无空章
- [ ] 附录包含术语与图表索引

## C. 模块写法正确性
- [ ] 平台管理模块使用“角色+触发+流程+状态+结果”
- [ ] 仅在算法/处理链路使用“输入/输出/执行步骤”

## D. 图表与编号
- [ ] 图表数量达到最低要求
- [ ] 图号/表号连续且唯一
- [ ] 正文引用与图表对应

## E. 格式一致性
- [ ] 标题层级规范
- [ ] 图注样式统一
- [ ] 目录层级与正文一致
- [ ] 术语命名前后一致

## F. 输出一致性
- [ ] Markdown 版本完整
- [ ] DOCX 版本结构与编号一致
