## Installation

### From Skills Collection

```bash
cd /path/to/claude-skills-collection
./scripts/install.sh citation-grabber
```

This will:
1. Clone the full repository
2. Install dependencies
3. Set up the skill for your AI CLI tool

## Standalone Installation

For the complete, standalone version with all features:

```bash
git clone https://github.com/yf8578/citation-grabber.git
cd citation-grabber
pip install -r requirements.txt
python3 citation.py "Your Paper Title"
```
